## 📞 Support During Deployment

**If logo doesn't show after upload:**
1. Check file name is exactly `logo.gif` (case-sensitive on Linux)
2. Check it's in ROOT folder (same level as index.html)
3. Hard refresh browser: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
4. Wait 5 minutes for Vercel cache to clear

**If getting 404 errors:**
1. Verify all 13 HTML files are uploaded
2. Check file names match exactly
3. All files must be in ROOT folder
4. Hard refresh browser

**If payment page doesn't load:**
1. Make sure `payment.html` is uploaded
2. Make sure `payment-success.html` is uploaded
3. Check they're in ROOT folder (not in subfolders)
